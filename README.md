# recriando-interface-netflix
Recriando a interface do principal site de streaming mundial utilizando tecnologias simples como HTML5, CSS3 e JavaScript. Conteúdo gerado por Felipe Aguiar da plataforma DIO, curso feito durante o _Bootcamp_ oferecido pela _Take Blip_.

* Inicialmente criou a paágina em HTML, para assim estilizá-la com o CSS3 no arquivo **main.css**.
* Foi usade de recursos externos como **Owl Carousel* e *fontawesome*.
* Foi colocado responsividade no modelo e exercitado a semântica presente no HTML.

Resumindo, foi uma experiência capaz de aperfeiçoar os conhecimento em HTML e CSS. Algumas modificações foram feitas em relação ao modelo do curso para assim parecer mais com o modelo atual da Netflix, contudo se reconhece a necessidade de aprofundamento para assim criar modelos mais complexos e interativos.

(Algumas modificações foram realizadas com a temática do **Dia internacional da Mulher** dando destaque aos filmes com mulheres protagonizando sua força. Para tornar o padrão comum, basta modificar as cores para as originais da Netflix, sendo elas #141414 para preto e #ffffff para branco)
